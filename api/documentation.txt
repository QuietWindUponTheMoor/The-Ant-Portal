Request types:

-------------------------------------
***post***
?type=post <-- Gets posts of any type, max of 100, default is max of 10

TO ASSIGN POST TYPE:
&post_type=<int> <-- 1 = question, 2 = sighing, 3 = nuptial flight

TO ASSIGN MAX POST LENGTH:
&max_length=<max> (&max_length=50 would grab 50 posts)

TO SET ASC OR DESC:
&order=1 (ASC)
&order=2 (DESC)

-------------------------------------
